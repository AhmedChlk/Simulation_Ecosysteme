# Simulation d’Écosystème — C++

Projet de programmation orientée objet réalisé en C++.

La simulation représente un monde en grille composé de tuiles dans lequel évoluent :

- de l’herbe (ressource),
- des herbivores,
- des dangers environnementaux (pièges, virus).

Le projet met en œuvre les concepts de la POO :

- abstraction et encapsulation,
- héritage et polymorphisme,
- STL et gestion de la mémoire,
- surcharge d’opérateurs.

L’affichage est réalisé avec la bibliothèque **SFML**.

---

## Prérequis

- **CMake**
- **SFML 2.5** (`graphics`, `window`, `system`)

### Installation (Linux / Ubuntu)

```bash
sudo apt install cmake libsfml-dev
```

### Compilation et exécution

```bash
make run
```
